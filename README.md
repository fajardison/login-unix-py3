## Install Tools In Termux  
- `pkg update && pkg upgrade`  
- `pkg install python`  
- `pkg install git`  
- `git clone https://github.com/fajardison/login-unix-py3.git`  

## Instalasi  
1. `cd login-unix-py3`  
2. `python setup.py`  
3. `cd ~`  
4. `source ~/.bash`  
5. `install`  
6. `run`
